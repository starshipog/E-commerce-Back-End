# E-commerce-Back-End

## Description

My motivation to build this application was to build a site for the growing world of e-commerce.
I built this project so business owners can have a clean website to sell there products or services.
I solved the problem of helping business owners setup a easy website for there stores.
I learned how to tie related information together using primary keys
